Preview Your Digital Products for Your Future Customers

== Description ==

LivePreview is a WordPress plugin that lets you manage and switch between your digital products like HTML themes & templates, HTML5 games and etc. The script include a responsive test tool inside the demobar, switch panel with thumbnails of the products, buttons for different actions (buy, download, iframe close). The plugin can be deployed easily. It runs on all modern browsers and mobile devices like iOS, Android and Windows.

This is the **LITE version** of the official [LivePreview - Digital Product Demo Bar for WordPress](https://1.envato.market/g7zKO) plugin which comes with support and doesn't have some limitations.


= The sample of how to create a livepreview item =
https://youtu.be/FBprXBIgMHo


= Features =
* **Powerful Interface** - many options & capabilities
* **3 Predefined Themes** - included 2 skins: top & bottom (you can add your own)
* **Responsive** - test your theme with different screen sizes
* **AJAX saving** - save your config without page reloading
* **Customization** - create you own theme


== Installation ==
* From the WP admin panel, click "Plugins" -> "Add new"
* In the browser input box, type "LivePreview"
* Select the "LivePreview" plugin and click "Install"
* Activate the plugin

Alternatively, you can manually upload the plugin to your wp-content/plugins directory

= What is the difference between Lite and PRO =
The lite version has only one limits:
1) You can create and use only 3 items
2) You can't save & load config from a json file
All other features are the same as PRO has.
